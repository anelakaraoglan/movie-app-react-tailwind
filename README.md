# 🎬 Movie App (React + Vite + Tailwind)

A simple movie application built with React, Vite, and Tailwind CSS.
Users can browse movies and interact with the interface in a clean, modern layout.

## 🚀 Features

* Display movies
* Responsive design
* Modern UI with Tailwind CSS

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript

## ▶️ How to Run

npm install
npm run dev

Open in browser: http://localhost:5173
